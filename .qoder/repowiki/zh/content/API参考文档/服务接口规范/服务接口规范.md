## 简介

开发者工具箱是一个基于 Electron 的桌面应用程序，提供了丰富的开发工具服务。本文档详细说明了主进程服务模块的接口定义，包括代码运行器、HTTP 客户端、SQL 专家、Dock 服务、域名查询、OSS 管理、NPM 管理和通知服务等模块的 API 规范。

该工具箱旨在为开发者提供一体化的开发环境，通过主进程服务模块实现各种功能，包括代码执行、网络请求、数据库查询、文件管理等。所有服务都通过 IPC 机制与渲染进程进行通信，确保了安全性和稳定性。

#### 初始化参数

代码运行器服务的初始化过程包括：

1. **模块劫持**：全局劫持 `http`、`https`、`net` 模块，跟踪服务器实例
2. **沙箱环境**：创建隔离的执行环境，限制访问权限
3. **安全模块**：实现受控的模块加载机制

#### 方法调用规范

| 方法 | 参数 | 返回值 | 描述 |
|------|------|--------|------|
| `run` | `code: string`, `language: 'javascript' \| 'typescript'` | `Promise<RunResult>` | 执行代码并返回结果 |
| `stop` | 无 | `void` | 停止当前执行的代码 |
| `clean` | 无 | `Promise<boolean>` | 清理所有活动的服务器实例 |
| `killPort` | `port: number` | `Promise<{ success: boolean; message: string }>` | 终止占用指定端口的进程 |

#### 错误处理

代码运行器实现了多层次的错误处理机制：

```mermaid
flowchart TD

CreateSandbox --> TransformCode["转换代码(如需)"]
TransformCode --> ExecuteCode["执行代码"]
ExecuteCode --> CheckResult{"检查执行结果"}
CheckResult --> |成功| FormatOutput["格式化输出"]
CheckResult --> |异常| HandleError["处理异常"]

ReturnSuccess --> End([执行结束])

```

## 性能考虑

### 代码运行器性能优化

代码运行器采用了多项性能优化措施：

1. **沙箱隔离**：使用 `vm.createContext` 创建隔离的执行环境
2. **超时控制**：设置 30 秒的执行超时时间
3. **内存管理**：及时清理活动的服务器实例
4. **输出限制**：限制输出内容的大小和格式

### HTTP 客户端性能

HTTP 客户端具有以下性能特点：

1. **连接复用**：使用 Electron 的 `net` 模块进行高效的网络请求
2. **超时处理**：支持可配置的超时时间
3. **错误恢复**：自动处理网络错误和超时情况

### SQL 专家性能

SQL 专家服务的性能优化包括：

1. **连接池管理**：使用 MySQL 连接池减少连接开销
2. **查询限制**：默认限制查询结果为 10 行
3. **缓存机制**：缓存数据库表结构和配置信息
4. **流式处理**：支持 AI 响应的流式处理

### 文件上传性能

OSS 上传服务采用以下优化策略：

1. **分片上传**：使用 5MB 的分片大小
2. **并发上传**：支持 4 个并发上传任务
3. **进度监控**：实时监控上传进度
4. **断点续传**：支持中断后的续传

## 故障排除指南

### 常见问题及解决方案

#### 代码运行器问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 代码执行超时 | 代码包含无限循环或长时间阻塞操作 | 检查代码逻辑，使用异步操作 |
| 模块加载失败 | 未安装所需的 NPM 包 | 在 NPM 面板中安装相应包 |
| 端口占用 | 之前的服务器实例未正确关闭 | 使用 `code:clean` 或 `code:killPort` 清理 |

#### HTTP 客户端问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| CORS 错误 | 跨域请求被阻止 | 使用 HTTP 客户端服务绕过限制 |
| 超时错误 | 网络连接缓慢 | 增加超时时间或检查网络连接 |
| 代理问题 | 代理设置不正确 | 在应用设置中配置正确的代理 |

#### SQL 专家问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 数据库连接失败 | 连接信息错误或网络问题 | 检查数据库配置和网络连接 |
| 查询超时 | SQL 查询过于复杂 | 优化 SQL 语句或添加索引 |
| AI 服务错误 | API 密钥无效或配额不足 | 检查 API 配置和账户状态 |

#### OSS 上传问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 上传失败 | 权限不足或配置错误 | 检查 OSS 配置和权限设置 |
| 上传超时 | 网络连接不稳定 | 检查网络连接或使用断点续传 |
| 进度不更新 | 网络延迟或服务器问题 | 检查网络状态或稍后重试 |
